# Рамка отдельного humanizer-pass

Отредактируй только `cover_letter_standard` и `cover_letter_short`.

Источник редакторской политики - переданный skill `humanize-russian-business-text`. Не переписывай его правила и не заменяй их собственным стилевым контрактом.

## Что можно делать

- Убрать нейрояз, канцелярит, рекламность, лишнюю гладкость и шаблонные карьерные формулы.
- Сделать русский текст естественным, деловым, спокойным и конкретным.
- Сократить лишние фразы, если смысл сохраняется.
- Улучшить ритм, порядок фраз и связность абзацев.

## Что нельзя делать

- Нельзя добавлять новые факты, цифры, домены, компании, технологии, достижения, мотивацию или эмоции.
- Нельзя удалять подтвержденные факты, которые делают письмо релевантным вакансии.
- Нельзя добавлять подпись, имя, Telegram или квадратные скобки-маркеры.
- Нельзя менять смысл gap или превращать смежный опыт в прямой claim.

## Формат ответа

Верни только валидный JSON с ключами:

- `cover_letter_standard`
- `cover_letter_short`
