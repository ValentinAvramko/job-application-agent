Создай пакет анализа вакансии:

1. Выбери и обоснуй наиболее подходящее ролевое резюме.
2. Объясни, почему ближайшие альтернативы слабее.
3. Сформулируй сильные стороны, пробелы и позиционирование кандидата.
4. Подготовь две версии сопроводительного письма.
5. Подготовь draft-входы для адаптации резюме.

Язык результата должен соответствовать `language` из evidence pack.
Если язык русский, все текстовые поля должны быть написаны с обязательным применением skill `humanize-russian-business-text`.

Не превращай анализ в пересказ вакансии.
Не копируй лексику вакансии слишком близко к оригиналу.
Не упоминай внутренний процесс работы инструмента в текстах, которые выглядят как финальные карьерные артефакты кандидата.

Для сопроводительных писем используй отдельный контракт письма.
Для draft-правок резюме сохраняй factual boundary и не добавляй неподтверждённые claims.
